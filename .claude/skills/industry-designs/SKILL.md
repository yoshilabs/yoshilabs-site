## How to Use

1. Identify the client's vertical from their brief
2. Load the corresponding model file from `models/{vertical}.yaml`
3. Apply the color system as CSS custom properties
4. Use the typography pairing for all headings and body text
5. Follow the section structure for that vertical
6. Apply the recommended micro-interactions
7. Use the CTA strategy for conversion optimization

## Philippine Context (ALL verticals)

Every site must include:
- GCash/Maya payment badges
- Philippine phone format (+63)
- Barangay or district reference
- WhatsApp/Viber/Messenger contact options
- Facebook Page integration (most PH SMEs have one)
- Mobile-first (80%+ PH traffic is mobile)
- Fast loading on 4G connections
- DTI/SEC registration if applicable
