Hero Stage — Proof Library

Seventeen brands, each rendered against a different hero_stage preset from the v2 schema. Sixteen are fictional proofs, one is Meadow — the real mymind-design skill serving as the calm pole. Together they prove the layered composition model: background field, optional hero subject (categorized by intent, not form), and relation.

Schema shape

hero_stage: preset: luminous-on-gradient # one of 9 v2 presets, or null background: medium: gradient # gradient / mesh / painterly / shader / pattern / bokeh / sculptural / noise / photo / absent color_mode: palette # monochrome / dual-tone / palette / brand-tinted-neutral saturation: vibrant # flat / muted / vibrant / neon light_source: center # top / bottom / tl..br / center / ambient / none vignette: subtle # off / subtle / strong intensity: bold # subtle / bold / blown-out — default subtle motion: static # static / drift / pulse / reactive color_palette: [...] # 3-5 hues hero: subject: luminous # none / luminous / object / device / composition / photo-cutout ← by intent form: sphere # sphere / disc / ring / torus — ONLY for luminous. Ignored for everything else. placement: center # center / off-center-l/r / bottom / top-corner-l/r scale: balanced # dominant / balanced / accent tint: gradient-from-palette # brand / neutral / gradient-from-palette / metallic relation: type: glow # flat / glow / halo / reflection / emissive / shadow-only (see compat matrix) bleed: 60 # 0-100 — luminous bleeds light, object does not # Honesty rule: luminous/device/composition render in CSS. object = generic warm metallic form # (decorative placeholder; user swaps for real 3D render). photo-cutout = prose placeholder.
