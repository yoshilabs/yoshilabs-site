Find

Remember one word. Find everything.

Search works the way your memory does — fuzzy, partial, associative. Type "the blue one from winter" and it surfaces. Search by color, by object, by feeling, by fragment.

  • Visual search — describe a color, a shape, a subject
  • Associative search — "that thing about maps" works
  • No query language — write the way you think
